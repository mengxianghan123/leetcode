## 1.题目
给定一个二叉树，返回所有从根节点到叶子节点的路径。  
说明: 叶子节点是指没有子节点的节点。  

示例:  
```
输入:  

   1
 /   \
2     3
 \
  5

输出: ["1->2->5", "1->3"]
解释: 所有根节点到叶子节点的路径为: 1->2->5, 1->3
```

---

## 2.分析
此题其实不难，准确说是很简单。。但是由于我从未处理树相关题目，写起来还是十分陌生。  
注意一个地方，我将树的初始化部分写进了一个函数，然后，出现了bug。这里我觉得是因为，函数中的变量是局部变量（包括树结构体类型的变量），函数调用结束释放栈空间，此时，root节点的左右子节点指针悬挂。
